Bienvenue sur VitaSequor

Votre application dédiée au personnel d'une clinique afin de gérer l'institution médicale du coté administratif, financier et médical.

Fonctionnalités par rôle

Admin principal

  • Créer et gérer les utilisateurs
  • Gérer les départements
  • Voir toutes les statistiques
  • Superviser le système

Admin de département

  • Gérer les utilisateurs du département
  • Enregistrer les spécialités
  • Optimiser l'organisation du département

Médecin

  • Voir ses rendez-vous
  • Consulter les dossiers patients
  • Prescrire des actes médicaux
  • Suivre l'historique médical

Secrétaire

  • Enregistrer les patients
  • Créer des rendez-vous
  • Gérer les dossiers patients
  • Organiser le planning
